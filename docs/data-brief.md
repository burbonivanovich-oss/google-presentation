## Общие соглашения по формату (как в `data(22).xlsx`)

- **1-я строка листа** — применённые фильтры (текстом), как сейчас:
  `Примененные фильтры: type_plan является A; scenario является Афина; …`
- **Шапка таблицы** — со 2–3-й строки, ровно теми названиями колонок, что ниже.
- **Период** — отдельной колонкой `Месяц` в формате даты `ГГГГ-ММ-01`
  (как в data(22): `2026-01-01 00:00:00`).
- **Числа** — без текстовых суффиксов (не «19 000 000 ₽», а `19000000`).
- Один файл = одна выгрузка (лист `Sheet1`), `.xlsx`.

Базовый набор фильтров продаж (как в data(22)) — сохраняем единым во всех
продажных выгрузках, чтобы цифры сходились с дашбордом:
`type_plan = A · scenario = Афина · Тип продажи = Подключение · Онлайн = Онлайн`

**Что чинит:** слайд «Q1 2026 по направлениям». Сейчас факт по направлениям —
оценка по доле БЮ, а **плана по направлениям нет вообще**.

- **Источник:** Царь свод все разрезы
- **Фильтры (1-я строка):**
  `type_plan = A · scenario = Афина · Тип продажи = Подключение · Онлайн = Онлайн`
- **Группировка строк:** `Бизнес-юнит` × `Месяц`

  Крупняк в ОФД* — то есть Розница / Общепит / Кассы / ОФД)
- **Колонки (точно):**
  `Бизнес-юнит | Месяц | Кол-во план | Оплата план | Кол-во факт | Оплата факт | Дельта шт | Дельта руб | % шт | % руб | Ср. чек план | Ср. чек факт | % ср. чек`
- **Период:** `2025-01` … `2026-12` (план — весь 2026; факт — по последний закрытый месяц)
